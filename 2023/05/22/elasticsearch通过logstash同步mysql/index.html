elasticsearch通过logstash同步mysql

发表于 2023-05-22 分类于 ES

说明

  • elasticsearch版本：7.4
  • logstash版本：7.4
  • mysql版本：8.0

logstash配置文件

devtools常用接口

#修改索引最大查询数

PUT es_index/_settings
{
“index”:{
“max_result_window”:1000000
}
}

#定义索引别名

POST /_aliases
{
“actions”: [
{
“add”: {
“index”: “es_index0114”,
“alias”: “es_index”
}
}
]
}

#es使用sql语法

POST /_sql
{
“query”: “select * from es_index where id = ‘26427’”
}

注意事项

  • logstash支持mysql视图查询，但数据过多时查询很慢，iops压力会很大，建议使用查询语句查询
  • logstash支持秒级查询，格式同crontab
  • mysql默认时间0000-00-00,非有效日期,logstash无法同步
